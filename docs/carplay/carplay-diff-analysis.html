Apple CarPlay — Analisi delle differenze implementative

RadioE45 · .NET MAUI 10 · Piattaforma iOS
3480aa5 · Initial Apple CarPlay and Android Auto support 43630a1 · HEAD (Fixed carplay support)

1 Flusso di navigazione CarPlay

Il cambiamento più visibile per l'utente: il template radice e la direzione della navigazione sono stati invertiti.

Impatto UX: Nella versione iniziale l'utente entrava subito nel NowPlaying (che mostrava uno schermo nero se niente era in riproduzione). Ora vede subito la lista stazioni — più intuitivo per un primo avvio CarPlay.

Nella versione iniziale il manifest era commentato e il routing scene non esisteva. Ora entrambi sono attivi.

4 Pipeline di build iOS

L'implementazione attuale aggiunge due MSBuild target per aggirare un limite di MAUI che rimuove UIApplicationSceneManifest durante il merge dei plist.

Prima — Entitlements.plist (eliminato)
<!-- com.apple.developer.carplay-audio
     richiede approvazione Apple.
     <key>com.apple.developer.carplay-audio</key>
     <true/>
-->
Dopo — CustomEntitlements in .csproj
<ItemGroup Condition="...== 'ios'">
  <CustomEntitlements
    Include="com.apple.developer.carplay-audio"
    Type="Boolean" Value="true"
    Visible="false" />
</ItemGroup>

5 Remote Command Center (lock screen / CarPlay)

Prima
// Una sola riga in FinishedLaunching:
application.BeginReceivingRemoteControlEvents();

// Nessun handler registrato →
// i tasti fisici e CarPlay non
// controllano la riproduzione
Dopo — SetupRemoteCommandCenter()
center.PlayCommand.Enabled = true;
center.PlayCommand.AddTarget(e => {
    audio.ResumeAsync();
    return Success;
});

center.PauseCommand.Enabled = true;
center.PauseCommand.AddTarget(e => {
    audio.PauseAsync();
    return Success;
});

center.TogglePlayPauseCommand.Enabled = true;
center.TogglePlayPauseCommand.AddTarget(e => {
    if (audio.IsPlaying) PauseAsync();
    else ResumeAsync();
    return Success;
});

// Stream live — disabilita tutto il seek:
center.SeekForwardCommand.Enabled = false;
center.SeekBackwardCommand.Enabled = false;
center.SkipForwardCommand.Enabled = false;
center.SkipBackwardCommand.Enabled = false;
center.NextTrackCommand.Enabled = false;
center.PreviousTrackCommand.Enabled = false;
Timing critico: SetupRemoteCommandCenter() viene chiamato dopo base.FinishedLaunching() così il container DI MAUI è già inizializzato e i servizi sono risolvibili.

6 Subtitle dinamico nelle card stazione

Prima — sempre statico
private CPListItem BuildStationItem(
    AzuraStation station, IAudioService audio)
{
    CPListItem item = new CPListItem(
        station.Name,
        station.Description);  // sempre fisso
    ...
}
Dopo — titolo brano se in riproduzione
private CPListItem BuildStationListItem(
    AzuraStation station)
{
    bool isCurrent =
        _audioService?.CurrentStation?.Id == station.Id;
    string? currentTitle =
        _nowPlayingService?.Current.Title;

    string? subtitle =
        (isCurrent && !string.IsNullOrEmpty(currentTitle))
        ? currentTitle          // ← titolo brano live
        : station.Description; // ← fallback

    CPListItem item = new(station.Name, subtitle);
    ...
}
